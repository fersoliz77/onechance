# One Chance — Landing Page UI Kit

## Overview
High-fidelity recreation of the One Chance marketing landing page.
Based on: `change mockup basicos/one_chance_landing.html`

## Components
- `Nav.jsx` — Top navigation bar with logo, links, auth buttons
- `Hero.jsx` — Asymmetric hero grid with player cards
- `Cards.jsx` — PlayerCard, CoachCard, MiniBadge
- `RoleStrip.jsx` — Role selector tags (Jugador / Técnico / Club / Representante)
- `Ticker.jsx` — Scrolling player ticker bar
- `index.html` — Full interactive landing page

## Design Width
1280px desktop. Cards use the full dark-navy + green brand palette.
